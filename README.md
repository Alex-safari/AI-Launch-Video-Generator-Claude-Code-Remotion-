# Overview

This repository contains a **Remotion-based video generation project** created using **Claude Code**, an AI coding agent by Anthropic.

Claude Code generates the entire video project from a single prompt.

Instead of manually editing videos inside tools like Premiere Pro, this system builds the video **programmatically using React and Remotion**.

Remotion renders the final result by:

1. generating every frame of the animation
2. capturing screenshots of each frame
3. stitching them into an **MP4 video**

The final output looks identical to a traditionally edited video вЂ” but it was created entirely with code.

---

# Core Idea

Two components make this system work:

## 1. Claude Code

Claude Code is an AI coding agent that can run inside:

- Terminal
- Cursor
- Claude Desktop
- Browser
- other agentic IDEs

It can read your codebase and generate or modify files automatically.

In this project, Claude Code generates the **Remotion video project** from prompts.

---

## 2. Remotion

Remotion is a framework that lets you **build videos with code instead of video editors**.

Instead of dragging clips onto a timeline, you define:

- scenes
- animations
- transitions
- text elements
- timing

Remotion then renders the final video.

---

# Remotion Skills for Claude Code

This project uses a **Remotion Skill** installed into Claude Code.

A **Skill** is essentially a cheat sheet that teaches Claude how to correctly use a tool.

The Remotion Skill includes:

- Remotion best practices
- timing rules
- animation patterns
- scene structure
- renderer limitations

Without the skill, Claude might generate incorrect Remotion code.

With the skill installed, Claude can generate working Remotion projects immediately.

Think of it like giving a contractor **blueprints before they start building**.
